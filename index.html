Поставленная задача

Написать конвертер игровых валют. В котором можно будет указать цену валюты и его количество и получить в ответ информацию, сколько игровой валюты можно выставить на продажу при наименьших затратах.

Решение задачи - описание работы

Программа для удобной оценки игровых ценностей, упрощающая оптовую торговлю. В игре Path of Exile активная торговая система между игроками, а также большое количество расходных и прочих предметов. При запуске пользователю необходимо выбрать или ввести:

  • Язык программы
  • Текущий игровой сезон.
  • Категорию игрового предмета
  • Относящийся к этому разделу предмет
  • Ввести количество предмета

Затем программа обратится к игровому API, получит необходимые данные, посчитает и выдаст результат.

Решение задачи - работа над проектом

Первая версия: Это был простой конвертер валют. Цены и количество конвертируемых валют вносились вручную для каждой операции.

Вторая версия: Я начал изучать модуль requests в Python и работу с API. Теперь программа сама обращалась на API игры и запрашивала список выставленных на продажу предметов и высчитав среднюю цену, на основе неё высчитывала результат. Но список предметов был ограничен вручную записанным в программу списком.

Третья версия: После обновления игрового API, программа была значительно переписана. Изменился алгоритм обращения к API и добавлен выбор языка программы. Было решено отказаться от заготовленного списка предметов, поскольку игра не стоит на месте и одни предметы заменяют другие. Для этого написал программу для парсинга торгового сайта игры и формирования списка разделов с предметами. Для этого изучил работу с Selenium.
